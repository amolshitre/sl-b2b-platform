Ant Script Library Download

The latest version is 1.0.1, which can be downloaded here:

http://www.exubero.com/asl/ant-script-library-1.0.1.zip

Alternatively, you can check out latest changes directly from the Subversion repository:

https://www.exubero.com/asl/svn/trunk/

Note that the SSL certificate on www.exubero.com is currently self-signed, meaning that you will probably be prompted about an invalid certificate. You will have to add an exception

Refer to the usage page for details on how to install the library.

Is there a problem or mistake on this page? Do you want to contribute some changes? Send me an email at joe@exubero.com.
