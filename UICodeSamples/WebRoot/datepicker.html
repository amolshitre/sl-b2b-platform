Datepicker

We use the datepicker provided in the jQueryUI library. The CSS theme we've selected for ServiceLive is called "Black Tie". See the How To section below about what scripts and stylesheets to import.

Date:

========

How to:

  1. Put these in the <head></head> section of your document: 
    <link rel="stylesheet" href="${staticContextPath}/css/jqueryui/jquery-ui-1.7.2.custom.css" type="text/css"></link>
<script type="text/javascript" src="${staticContextPath}/javascript/jquery/jquery-1.3.2.min.js"></script>
<script type="text/javascript" src="${staticContextPath}/javascript/jquery/jqueryui/jquery-ui-1.7.2.custom.min.js"></script>
    Note: Don't insert duplicates if the above files have already been referenced on the page. Also, the above code assumes that you've defined the "staticContextPath" variable in your JSP.
  2. Initialize the datepicker: 
    <script type="text/javascript">
	$(function() {
		$("#datepicker").datepicker();
	});
	</script>
  3. Create a field for the datepicker to update: 
    <form>
<p>Date: <input id="datepicker" type="text" readonly="readonly"></p>
</form>
    Note: The readonly attribute prevents user input into the field, but allows it to be updated programmatically. Remove it if the requirement is for users to be able to enter a date manually.
