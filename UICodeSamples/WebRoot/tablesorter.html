How to:

  1. Put these in the <head></head> section of your document: 
    <script type="text/javascript" src="${staticContextPath}/javascript/jquery/jquery-1.3.2.min.js"></script>
<script type="text/javascript" src="${staticContextPath}/javascript/plugins/jquery.tablesorter.min.js"></script>
    Note: The above code assumes that you've defined the "staticContextPath" variable in your JSP.
  2. Initialize the tablesorter object: 
    <script type="text/javascript">
	jQuery(document).ready(function($) {
		$("#sortedTable").tablesorter();
	});
	</script>
  3. Create a table to sort - the table must contain a thead and a tbody tag: 
    <table id="sortedTable"> 
<thead> 
<tr> 
    <th>Last Name</th> 
    <th>First Name</th> 
    <th>Email</th> 
    <th>Due</th> 
    <th>Web Site</th> 
</tr> 
</thead>
