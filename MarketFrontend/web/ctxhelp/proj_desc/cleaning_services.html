Tips for Project Description

Cleaning Services

Note: These questions may not apply to your project

  • What kind of product(s) is/are to be cleaned?
  • How would you describe a successful service visit?
  • Is the area to be cleaned indoors or outdoors? If indoors, which room(s)?
  • What are the dimensions of the area? (3'x5'x5', 12'x12')
  • What is the flooring type? (rug, hardwood floors, siding, etc.)
  • Can you describe the surface? (granite, marble, cherry wood, etc)
  • Are any special cleaning solutions required?
  • Is a ladder required?
  • Does any furniture need to be moved?
